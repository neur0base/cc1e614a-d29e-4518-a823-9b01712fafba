# コンポーネント名: MyTasks Header
コンポーネントの説明: アプリのヘッダーで、タイトルとタスク追加ボタンを表示します。
- **ID**: MyTasks_MyTasks_Header
- **名前**: MyTasks Header
- **パラメータ**: 未設定
- **コンポーネントタイプ**: コンテナ
- **モデル**: 未設定
- **コンポーネントデザイン**: ヘッダーには左側にアプリのタイトル 'MyTasks' が表示されます。右側には新しいタスクを追加するための '+' ボタンがあります。ボタンをクリックすると、タスク追加モーダルが開きます。ヘッダーは明るい背景色を持ち、タイトルとボタンが目立ちます。
- **モーダル表示**: いいえ
